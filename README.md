# 💰 FinWise – Behavioral Banking for Gen Z

**FinWise** is a novel, gamified banking management system that doesn't just *track* money—it *coaches* you. Built entirely in a single HTML file with pure JavaScript and LocalStorage, it actively fights impulsive spending, automates micro-savings, and visualizes your future wealth.

## 📌 Novel Problem Statement

**Title:** FinWise – A Behavioral Banking Engine for Curbing Impulsive Spending and Enabling Micro-Savings among Gen Z

**Problem Statement:** Traditional banking applications function as passive transaction ledgers that merely record deposits and withdrawals, offering no cognitive friction against impulsive spending or recurring subscription drains. FinWise solves this by introducing an automated 50/30/20 allocation engine, a painless "round-up" micro-vault, a subscription leech detector, and a gamified XP/leveling system—transforming passive banking into an active, habit-forming financial coach for digitally-native users.

---

## ✨ Unique Features

| Feature | Description |
| :--- | :--- |
| **50/30/20 Auto-Allocator** | When you add income, the system automatically splits it into Needs (50%), Wants (30%), and Savings (20%) *before* you spend a dime. |
| **"Round-Up" Micro-Vault** | Every expense is rounded up to the nearest dollar. The spare change (e.g., $0.50 from a $4.50 coffee) goes into a Vault. When it hits $10, lock it into Savings! |
| **Subscription Leech Detector** | Detects recurring payments (Netflix, Spotify, Prime, etc.) using keyword matching and calculates your shocking "Annual Leakage" cost. |
| **Compound Interest Visualizer** | Drag a slider to see how much your current savings will grow over 1 to 30 years at 10% CAGR. |
| **Gamification (XP & Badges)** | Earn XP for adding transactions, logging in daily, and staying disciplined. Unlock badges like *"Saver Novice"*, *"Subscription Slayer"*, and *"Investor in the Making"*. |
| **Zero Backend Required** | Runs entirely in your browser using **LocalStorage**. No database, no API keys, no setup. |
